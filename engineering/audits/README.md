# Audits

Dated output from running a prompt or scenario against the current codebase.
Name files `YYYY-MM-DD-<topic>.md` (e.g. `2026-07-21-qa-persona-sweep.md`).

Each entry should say what was checked, what was found, and whether it got
fixed in the same pass or is still open — this folder is a trail, not a todo
list, so don't leave "found but not yet triaged" items with no resolution
noted indefinitely.
